# ArsDashFunction

A function functional toolbelt for [azu/ArsScale](https://github.com/azu/ArsScale "azu/ArsScale").

## Installation

``` sh
pod 'ArsDashFunction'
```

## Usage

``` objc
ArsMin@[@1, @2, @3])// @1
ArsMax(@[@1, @2, @3])// @3

ArsRange(@1, @3); // @[@1,@2]
ArsRange(@3, @10, @3);// @[@3, @6, @9]
```
